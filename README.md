**アプリ名**<br>
お問合せ投稿&管理システム<br><br>
**作成した目的**<br>
ユーザーの意見を収集し、管理するシステムが必要だったため。<br><br>
**アプリケーションURL**<br>
お問合せ送信http://127.0.0.1:8000/info<br>
お問合せ管理http://127.0.0.1:8000/control<br><br>
**機能一覧**<br>
- お問合せの入力と送信
- 入力したお問合せ内容の修正
- ユーザーが問い合わせした内容の管理・検索<br>


**使用技術**<br>
Laravel 8<br><br>
